# Démonstration de la facilité de casser un mot de passe
Cette démonstration a pour objectif de démontrer la facilité et la rapidité de cassage de certains mots de passe trop commun ou trop simple. 

## Outils nécessaires
- hashcat
- un hash de mot de passe vulnérable
- un environnement Linux
- Une wordlist 

## Avertissement légal
Ne pas réaliser la démonstration sur un réel mot de passe. On peut prendre un mot de passe déjà présent dans la wordlist rockyou par exemple. On peut aussi combiner cette démonstration avec [Construction d'une wordlist de mots de passe pour une personne spécifique](../password-list-builder/) et utiliser la wordlist construite.
