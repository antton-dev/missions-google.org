# Objectif
Ce dépôt Github a pour objectif de regrouper l'ensemble de ma documentation technique personnelle et mes démonstrations réalisée dans le cadre de mes missions de sensibilisation à la cybersécurité. Ces missions sont réalisées dans le cadre de la subvention Google.org reçue par mon école Grenoble INP - Esisar pour sensibiliser les TPE/PME/associations locales.

Les démonstrations décrites dans le dépôt sont réalisées dans un environnement contrôlé, local et/ou avec autorisation écrite et détaillée.
